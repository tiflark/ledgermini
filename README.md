Краткое описание проекта

LedgerMini — это кроссплатформенное настольное бухгалтерское приложение для ведения учета по РСБУ (Российским стандартам бухгалтерского учета). Система реализует принцип двойной записи, позволяет вести план счетов, контрагентов, формировать проводки и основные отчеты. Проект имеет модульную архитектуру и графический интерфейс.

Технологический стек и используемые инструменты

Язык: C++17 (стандарт)
Фреймворк для GUI и работы с БД: Qt 6

Модули: Core, Widgets, SQL, PrintSupport
База данных: SQLite 3 (встраиваемая, файловая)
Система сборки: CMake
Контейнеризация: Docker / Docker Compose (для создания переносимого образа)
Система контроля версий: Git (репозиторий на GitHub)
Операционная система для разработки/запуска: Ubuntu Linux

Ключевая функциональность

План счетов: Иерархическое дерево счетов РСБУ (активные, пассивные, активно-пассивные).
Контрагенты: Ведение справочника организаций и ИП с ИНН/КПП.
Бухгалтерские проводки: Форма для ввода операций по дебету/кредиту с указанием суммы, даты, документа и контрагента.
Карточка счета: Детализация всех операций по выбранному счету за период.
Оборотно-сальдовая ведомость (ОСВ): Основной бухгалтерский отчет за выбранный период с начальным и конечным сальдо, оборотами.
Журнал операций: Просмотр всех проводок с расширенным фильтром (по дате, сумме, счету, контрагенту) и экспортом в PDF/CSV.
Поиск и фильтрация: Универсальный фильтр для проводок.
Экспорт: Возможность вывода отчетов и таблиц в PDF и CSV форматы.
Статус проекта

Рабочий прототип (MVP). Реализованы все основные функции для учета, интерфейс на русском языке. Проект успешно собирается с помощью CMake в Linux, упакован в Docker-образ, размещен на GitHub.
